# v0.1.0

`PM Method Agent` 的首个公开版本。

这一版的定位不是完整产品，而是一个已经可以被体验、被讨论、被继续迭代的方法内核验证版。

## 当前已具备

- 一套面向问题定义质量的方法内核
- 一版最小可运行的 CLI 入口
- 三个核心分析模块
- 稳定的结构化审查卡输出
- 场景基础信息输入层
- 示例案例、体验用例和最小测试
- 面向后续 CLI / 网页 / agent 共用内核的架构预留

## 当前重点

这一版重点验证的是：

- 输出结构是否稳定
- 阶段推进是否合理
- 场景补充、决策关口、审查卡三类输出是否顺手
- ToB / ToC / PC / 移动端语境是否已有基本差异

## 当前还没做

- 完整多轮会话能力
- 正式网页入口
- 开放 API
- 复合多代理运行时
- 更完整的评测案例集

## 推荐体验方式

当前最稳的体验方式：

```bash
PYTHONPATH=src python3 -m pm_method_agent.cli \
  --business-model tob \
  --primary-platform mobile-web \
  --target-user-role 前台 \
  --target-user-role 诊所管理者 \
  --product-domain 医疗服务平台 \
  "前台希望增加一个预约前提醒弹窗，避免漏提醒患者。"
```
